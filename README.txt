Deploy to Netlify (Free):
1) Go to https://app.netlify.com/ and create an account.
2) Drag & drop the 'phoenixofresilience_website' folder into the Sites area OR connect via GitHub.
3) Add your custom domain phoenixofresilience.org and enable HTTPS (Let's Encrypt).
4) The contact form is configured with data-netlify="true".
5) The Apply page embeds your Google Form (viewform).

Edit content:
- HTML files in root
- CSS in assets/css/style.css
- Images in assets/img/

Enjoy!
